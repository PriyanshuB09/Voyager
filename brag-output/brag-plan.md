# Brag Plan: Voyager

## What is this app?
Voyager is an open-source FIRST Robotics Competition autonomous builder that turns field waypoints and branching logic into deployable robot routines, with live NT4 position feedback.

## The angle
Treat autonomous programming like mission control: a route draws itself across the real FRC field, logic branches assemble beside it, and the result moves from plan to live robot to published routine. The flex is visual clarity—complex autonomy becomes something a team can see, shape, and ship.

## Hook (first 2-3 seconds)
The FRC field emerges from darkness under a precise scan line. The line “Autonomy should be visible.” locks in while a teal route begins drawing across the field.

## Key moments (the middle)

- Real Voyager “Auto maker field” UI with waypoints, rotation targets, an event trigger, and a constraint zone arriving along a route.
- The route expands into Voyager’s branching path tree: Path, If, Loop, Event, and Interruptable blocks connect into a routine.
- A live robot marker moves across the field while NT4 status flips from offline to connected, followed by “Publish Routine.”

## Outro / punchline
Voyager lands full-frame with the line: “Build the route. Branch the logic. Ship the auto.”

## User flow worth showing
Create a field path → compose branching autonomous logic → watch the live robot position → publish the selected routine.

## Tone
- Preset: cinematic
- Creative direction: FRC mission control, precise rather than bombastic
- Interpretation: Large declarative type, confident camera moves, real interface detail, and a restrained technical pulse; the product does the showing.
